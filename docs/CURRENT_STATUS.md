# Current Status

마지막 정리일: 2026-05-06

## 지금 되는 것

- 앱은 Expo + React Native 기반이다.
- 일정, Task, 메모, 탭별 카테고리 관리를 지원한다.
- Today/List 화면에서 D-day / Count, 일정, 반복 일정, Task, 반복 Task를 구분해 보여준다.
- 체크 패널은 Task / 마감일 / Count 흐름으로 나뉜다.
- 반복 일정은 범위 수정 정책을 따르며, 종료일 있는 반복과 종료일 없는 반복을 지원한다.
- 일정 편집에서 시작일과 반복 종료일을 설정할 수 있다.
- 반복 종료일은 날짜 지정, `계속`, `없음`을 구분한다.
- `계속`은 종료일 없는 반복으로 저장한다.
- 시작 시간과 종료 시간을 설정할 수 있고, 종료 시간은 `없음`으로 비울 수 있다.
- 알림 없음, 정시, 사전 알림 시간을 지원한다.
- Android 위젯용 네이티브 코드가 포함되어 있다.

## 이번 작업에서 정리한 것

- 반복 종료일 영역에 `계속` 버튼을 추가했다.
- 반복 중 종료일이 비어 있으면 `계속 반복`으로 보이게 했다.
- `계속` 선택 시 반복 종료일은 비우고 반복 타입은 유지되게 했다.
- 웹에서 맞춘 `계속` 반복 의미와 앱 저장 의미를 맞췄다.

## 확인한 것

- `npm run harness:recurring-scope` 통과.
- `npx expo export --platform android` 통과.
